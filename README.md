# 🐾 Sistema de Adoção de Animais

## 📌 Sobre o projeto
Plataforma web desenvolvida para gerenciar e divulgar animais disponíveis para adoção, permitindo que usuários se cadastrem, visualizem animais e realizem solicitações de adoção.

## 🚀 Funcionalidades
- Cadastro e login de usuários (autenticação com JWT)
- CRUD de animais com upload de imagens
- Filtros por espécie, porte, idade e raça
- Solicitação de adoção e acompanhamento
- Painel administrativo

## 🛠️ Tecnologias utilizadas
- Front-end: React
- Back-end: Node.js, Express
- Banco de dados: MySQL / MongoDB
- Autenticação: JWT
